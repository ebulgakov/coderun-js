Своя реализация Promise.all

Описание:

Вам нужно написать функцию myPromiseAll, которая полностью повторяет поведение стандартного Promise.all.

Требования:

  1. Функция принимает итерируемый объект (для простоты остановимся на массиве), который может содержать как промисы, так и обычные значения (примитивы, объекты).
  2. Возвращает новый Promise.
  3. Если все переданные промисы успешно завершились (fulfilled), возвращаемый промис должен зарезолвиться массивом их результатов. Порядок результатов в массиве должен строго соответствовать порядку исходных промисов в переданном массиве, а не времени их выполнения.
  4. Если хотя бы один из переданных промисов завершился с ошибкой (rejected), возвращаемый промис должен немедленно зареджектиться с этой же ошибкой.
  5. Если передан пустой массив, функция должна сразу же зарезолвиться пустым массивом.
